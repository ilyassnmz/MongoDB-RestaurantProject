# Foodu - ASP.NET Core & MongoDB Restoran Yönetim Sistemi

![Foodu Logo](https://hizliresim.com/4zbenhr)

Foodu, modern restoranların dijital ihtiyaçlarını karşılamak üzere geliştirilmiş, ASP.NET Core 10.0 ve MongoDB tabanlı kapsamlı bir restoran yönetim sistemidir. Proje, hem müşteri odaklı halka açık web sitesi hem de yöneticiler için güçlü bir admin paneli içermektedir.

## 🚀 Özellikler

### 👥 Halka Açık Web Sitesi
- **Kategori Bazlı Ürün Listeleme** - Ürünleri kategorilere göre filtreleme
- **Şef Tanıtım Sayfaları** - Restoran ekibini tanıtma
- **Restoran Galerisi** - Mekan ve yemek fotoğrafları
- **Müşteri Yorumları** - Değerlendirme ve yıldızlama sistemi
- **Online Rezervasyon** - Tarih ve saat seçerek masa ayırtma
- **E-bülten Aboneliği** - Kampanya ve duyurular için abonelik
- **Blog Sayfası** - Güncel haberler ve yazılar
- **Tamamen Responsive Tasarım** - Mobil, tablet ve desktop uyumlu

### 🔒 Admin Paneli
- **Dashboard** - Özet istatistikler ve metrikler
- **Ürün Yönetimi** - CRUD işlemleri, kategori yönetimi
- **Kategori Yönetimi** - Kategori ekleme, silme, güncelleme
- **Şef Yönetimi** - Şef bilgileri ve fotoğraf yönetimi
- **Blog Yönetimi** - Yazı ekleme, düzenleme, silme
- **Galeri Yönetimi** - Görsel yükleme ve düzenleme
- **Rezervasyon Yönetimi** - Bekleyen/Onaylanan/İptal edilen kayıtlar
- **Mail Gönderme Sistemi** - Abonelere toplu indirim maili gönderme
- **İndirim Kodu Oluşturma** - Özel kampanya kodları
- **Abone İstatistikleri** - Aktif/toplam abone sayıları
- **Cookie Tabanlı Güvenli Giriş** - Admin paneli yetkilendirme

## 🛠️ Kullanılan Teknolojiler

### Backend
- **ASP.NET Core 10.0** - Ana uygulama çatısı
- **MongoDB** - NoSQL veritabanı yönetimi
- **Repository Pattern** - Veri erişim soyutlaması
- **Mapster** - Entity ↔ DTO nesne eşleştirme
- **MailKit (SMTP)** - Toplu mail gönderme işlemleri
- **Cookie Authentication** - Güvenli kimlik doğrulama
- **FluentValidation** - Sunucu taraflı validasyon

### Frontend
- **HTML5 & CSS3** - Modern web standartları
- **Tailwind CSS** - Utility-first CSS framework
- **Bootstrap 5** - Responsive grid ve bileşenler
- **JavaScript** - Dinamik kullanıcı etkileşimleri
- **Material Icons** - Zengin ikon kütüphanesi
- **Google Fonts** - Jost, Oswald font ailesi
